BillMate adalah aplikasi web kalkulator patungan makan kelompok 
(split-bill)dinamis yang dirancang untuk menghitung pembagian tagihan secara otomatis, adil, 
dan proporsional. Dibangun menggunakan teknologi HTML, JavaScript Vanilla (DOM 
Manipulation), dan Tailwind CSS di sisi Front-End, serta menggunakan sistem komunikasi 
HTTP Fetch standar ke Back-End, aplikasi ini berfokus pada kemudahan penggunaan 
kelompok 
tanpa perlu sistem login/autentikasi yang rumit. Komponen visualnya mengusung tema modern 
dengan warna dominan jingga hangat (orange candy) dan tipografi Pop
